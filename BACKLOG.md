## 🏗️ ④ 카드/조합 대공사 (다음 세션 집중) — 설계 확정본

### 현재 상태 (이미 있는 것)
- 카드 = **2축**: 레어도(N/R/SR, 카드별 고정) × 강화별(★1~5, 머지로 상승)
- **8종**: 노력파(N,전능+15) / 스탯6종(R,해당스탯+55) / 천재(SR,전능+35). **SSR·UR은 정의만 있고 카드 없음**(향후 조합용으로 비워둠)
- **강화(머지) = 이미 구현**: 같은 카드 같은 별 2장 → ★+1, **확률적**(🌟대성공=1장보존 / ✨성공 / 💦실패=1장소실 / 💥대실패=2장소실), 별 오를수록 성공률↓(92→78→62→46%)
- 획득: 육성 완주 드랍 + 은퇴 시 카드
- 효과: 육성 시작 스탯 보정(장착 최대 2개, 시작 시 1장 소모, 완주 시 재획득)
- **문제**: UI가 리스트+"합치기" 버튼뿐(슬롯·연출 없음) / SR이 천장(SSR·UR 도달 불가) / 카드 8종뿐

### 목표 3가지
1. **합성 축 신설** — 레어도를 올리는 새 조합(SR 천장 돌파 → SSR/UR). + 결과물 새 카드로 종류 확장
2. **조합 화면 재설계** — 리스트+버튼 → **슬롯 2개 + 조합 버튼 + 겹침 애니 + 결과별 연출**. 강화·합성 한 화면 통합
3. **보관함 ↔ 조합 분리** — 보관함(보유·도감) / 조합(작업대) 화면 나눔

### 설계안 (내 제안 — ❓표시가 재율애비 결정 포인트)
- **조합 슬롯 2개, 놓는 카드로 자동 판별**:
  - 같은 카드+같은 별 → **강화**(별↑, 기존 로직 재활용)
  - 다른 카드, 같은 레어도 → **합성**(레어도↑, 신규)
- **✅결정1 — 합성 규칙 = 레어도 사다리**: 같은 레어도 아무 2장 → 상위 레어도 1장. `R+R→SR`, `SR+SR→SSR`, `SSR+SSR→UR`. 확률적.
- **✅결정2 — 결과물 = 하이브리드 + 특수효과**: 상위 카드는 2스탯 동시 부스트(하이브리드) / 전능 / 특수효과(성장률·관문 등)로 성격 다양화 → 종류 자연 확장.
- **카드 사다리 초안** (수치·목록은 구현 때 확정):
  | 레어도 | 성격 | 예시 |
  |---|---|---|
  | N | 전능 소 | 노력파(전능+15) |
  | R | 단일스탯 강 | 황금성대(발성+55) 등 6종 |
  | SR | 2스탯 하이브리드 | R+R 합성 결과. 발성+리듬="보컬래퍼" / 기존 천재(전능+35)도 SR |
  | SSR | 강한 하이브리드/특수 | SR+SR 합성 |
  | UR | 전능·특수 끝판 | SSR+SSR 합성. 전능+70 / 성장률↑ / 관문완화 등 |
  - 하이브리드: 6스탯 중 2개 조합 → 최대 C(6,2)=15종 (대표 몇 개부터)
  - 특수효과 후보: 시작스탯 대폭 · 성장률↑ · 관문 완화 (⚠️골드·이벤트확률은 게임 경제에 그 시스템 있는지 구현 때 확인)
- **합성도 확률 유지**(강화와 톤 통일): 대성공→상위 확정+재료보존 / 성공→상위 / 실패→재료만 소실. 레어도 오를수록 리스크↑
- **연출**: 두 슬롯 카드가 중앙으로 빨려들어가 겹침 → 발광 → 결과 등급별(대성공 무지개·성공 반짝·실패 연기)

### 내 추천 없이 그냥 진행할 것들 (재율애비 뒤집고 싶으면 말해)
- 조합 UI = 슬롯2 통합 자동판별 / 획득처는 기존(육성드랍+은퇴) 유지 / 강화 상한 ★5 유지 / 합성도 확률제
